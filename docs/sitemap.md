## SEO-структура

### Текущие индексируемые страницы (MVP)
- `/` — «Автобусные туры из южных городов России»
- `/tours` — «Каталог автобусных туров — все направления»
- `/tours/arhyz-gornyj-weekend` — «Тур в Архыз на выходные из Краснодара»
- `/from/krasnodar` — «Автобусные туры из Краснодара — расписание и цены»

### Принципы формирования title/description
- **title**: `{Название тура} — от {цена} ₽ | ЮгТур`
- **description**: включает город выезда, направление, цену, даты
- Каждая страница имеет уникальные metadata

### Будущая автоматизация
- Страницы `/from/[city]` генерируются автоматически из данных
- Страницы `/destinations/[slug]` генерируются из направлений в mock data
- Сезонные подборки формируются по датам и тегам
